Trail Monkeys Rosenheim — QA Checklist
=======================================

Run these checks after every significant change and before publishing.
Start a local server first: python3 -m http.server 8000


GENERAL (all pages)
-------------------

[ ] Page loads without browser console errors (check DevTools > Console)
[ ] Page loads without CSS errors (check DevTools > Network)
[ ] <title> tag is present and unique per page
[ ] <meta description> is present and under 160 characters
[ ] Only ONE <h1> per page
[ ] Logo is visible in the nav bar
[ ] TSV badge is visible in the nav bar (top right)
[ ] All nav links are clickable and go to the correct page
[ ] Current page is marked with aria-current="page" in nav
[ ] Skip-to-content link appears on Tab key press
[ ] Footer links all work (no 404s)
[ ] Footer copyright text is present
[ ] No broken images (all imgs show content or meaningful alt text)


RESPONSIVE / MOBILE
-------------------

[ ] Resize to 360px width — content remains readable, no horizontal scroll
[ ] Nav collapses to hamburger menu at mobile width
[ ] Hamburger menu opens and closes correctly on tap/click
[ ] Hero text is readable at all sizes
[ ] CTA buttons are tap-friendly (min 44x44px touch target)
[ ] Gallery grid collapses to single column on mobile
[ ] Training group cards stack vertically on mobile


ACCESSIBILITY
-------------

[ ] Tab through entire page — focus outline is always visible
[ ] Tab order is logical (matches visual order)
[ ] All images have meaningful alt text (or alt="" for decorative)
[ ] Color contrast ratio ≥ 4.5:1 for body text (check with browser dev tools)
[ ] CTA buttons are keyboard-activated (Enter/Space)
[ ] Gallery items can be opened via keyboard (Enter/Space)
[ ] Lightbox: ESC key closes lightbox
[ ] Lightbox: arrow keys navigate prev/next
[ ] Lightbox: focus is trapped inside lightbox while open
[ ] Lightbox: focus returns to trigger element on close
[ ] aria-current="page" set on active nav link
